# 🛍️ Shopping Data Cleaning with Pandas

## 🌸 Project Overview
This repository showcases my assignment on **Basic Data Exploration and Cleaning using Pandas**.  
The goal was to take a raw shopping dataset and transform it into a clean, structured format that can be used for analysis.  
It’s a mix of coding, learning, and a little pastel‑aesthetic touch ✨

---

## 🎯 Objectives
- Learn Python basics with Pandas
- Perform data exploration (head, shape, columns, dtypes)
- Handle missing values (identify, fill/drop)
- Remove duplicates
- Clean price column (convert to numeric)
- Create a derived column (`calculated_final_price`)
- Save the cleaned dataset as a new CSV

---

## 🧩 Steps Completed
1. **Loaded Dataset** → `backpacks.csv` into Pandas DataFrame  
2. **Explored Data** → checked shape, columns, datatypes, head/tail  
3. **Handled Missing Values** → identified and dropped nulls  
4. **Removed Duplicates** → ensured unique rows  
5. **Cleaned Price Column** → removed symbols and converted to float  
6. **Derived Column** → calculated discounted price (`calculated_final_price`)  
7. **Saved Cleaned Dataset** → `backpacks_cleaned.csv`  

---

## 🌼 Brief Summary
In this assignment, I loaded the shopping dataset into a Pandas DataFrame and explored its structure using head, shape, and data types.  
I identified and handled missing values, removed duplicates, and cleaned the price column by converting it into numeric format.  
I also created a derived column (`calculated_final_price`) to check discounted prices.  
Finally, I saved the cleaned dataset as a new CSV file.  
This process helped me understand the basics of **data exploration and cleaning with Pandas** 🌸
